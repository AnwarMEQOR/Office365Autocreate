# Office365 Autocreate

Office365 Autocreate is a simple Python Code that automatically creates Office365 accounts.

## Requirements
#### Install Python:
[Python official website](https://www.python.org/downloads/)

#### Install Python Modules
```python
pip install selenium
```
```python
pip install names
```
You will also need: time. But it is preinstalled in most Python releases.
## Usage
After installing Python and the necessary modules above, follow the instructions bellow:
* Run the "Run.py" file.
* You will be asked how many accounts you would like to create.
* Press Enter.
* Wait for the code to generate accounts for you.
* Congratulations. Your Office365 accounts are ready to use. Log in to [Office365](https://www.office.com/) and change your password.
